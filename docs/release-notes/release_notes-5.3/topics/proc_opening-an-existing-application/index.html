Opening an existing application in MTA

You can open and edit an application that has already been defined in {ProductShortName}.

Prerequisites

  • You are logged in to an {ProductShortName} server.

Procedure

  1. In the {ProductShortName} user interface, select the Migration working mode.

  2. Click Application Inventory in the left menu bar. A list of the available applications appears in the main pane.

  3. Click edit ({edit}) to open and review the application settings.

    For the more information about the list of application settings, see Creating an application by using MTA.
