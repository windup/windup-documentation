Installing the {ProductShortName} extension for Visual Studio Code

You can install the {ProductShortName} extension for Visual Studio Code (VS Code).

Prerequisites

Unresolved directive in installing-vs-code-extension.adoc - include::snippet_jdk-hardware-mac-prerequisites.adoc[]

Procedure

  1. Set the environmental variable JAVA_HOME:

    $ export JAVA_HOME=jdk11

  2. In VS Code, click the Extensions icon on the Activity bar to open the Extensions view.

  3. Enter {ProductName} in the Search field.

  4. Select the {ProductName} extension and click Install.

    The {ProductShortName} extension icon () is displayed on the Activity bar.
