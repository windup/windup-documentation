Exporting an assessment questionnaire

In the {ProductName} ({ProductShortName}) 7.0, you can export an assessment questionnaire to the desired location on your system.

Procedure

  1. In the Administration view, select Assessment questionnaires.

  2. Select the desired questionnaire.

  3. Click the Options menu ({kebab}).

  4. Select Export.

  5. Select the location of the download.

  6. Click Save.
