Installing the {ProductShortName} plugin for IntelliJ IDEA

You can install the {ProductShortName} plugin in the Ultimate and the Community Edition releases of IntelliJ IDEA.

Prerequisites

Unresolved directive in installing-intellij-idea-plugin.adoc - include::snippet_jdk-hardware-mac-prerequisites.adoc[]

  • The latest version of {LC_PSN}-cli from the {ProductShortName} download page

Procedure

  1. In IntelliJ IDEA, click the Plugins tab on the Welcome screen.

  2. Enter {ProductName} in the Search field on the Marketplace tab.

  3. Select the {ProductName} ({ProductShortName}) by Red Hat plugin and click Install.

    The plugin is listed on the Installed tab.
