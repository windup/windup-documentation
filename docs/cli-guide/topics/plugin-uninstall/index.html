Uninstalling the {PluginName}

  1. In Eclipse, navigate to Help → Installation Details.

  2. In the Installed Software tab, select Windup Feature and Windup Feature Developer Resources.

    Press the Ctrl key to select multiple rows.

  3. Select Uninstall.

  4. Review the details and click Finish to uninstall the plugin.

  5. Restart Eclipse for the changes to take effect.
