Adding custom rules

You can add custom rules to use in each project. When you add a rule, it is disabled by default. If you no longer need a rule, you can delete it.

Procedure

  1. Click Add rule.

    1. In the Upload tab, click Browse and select a rule file (\*.windup.xml).

    2. In the Server path tab, enter the Rules path of the server-side file or directory containing the rule file in the field, select Scan recursively if relevant, and click Save.

  2. Click Next.
